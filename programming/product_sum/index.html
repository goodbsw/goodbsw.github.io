Product_sum

1. Code

type SpecialArray []interface{}

func ProductSum(array []interface{}) int {
    return helper(array, 1)
}

func helper(array SpecialArray, multiplier int) int {
    sum := 0
    for _, el := range array {
        if cast, ok := el.(SpecialArray); ok {
            sum += helper(cast, multiplier+1)
        } else if cast, ok := el.(int); ok {
            sum += cast
        }
    }
    return counter * sum
}

2. Conceptual Overview

Time Complexity: O(n)

  • It is really important to specify what n is. It is wrong just to say n is the number of element in the array passed in. The n is the total number of element including the subarray. Let’s say we have [1, 2, [3, 4], 5]. The total number of n is 6 not just 4. We have to consider subarray as well as the elements in the subarray

Space Complexity: O(d)

  • Since this algorithm is using recursive function, we have to take call stacks into consideration. We can decide how many call stacks by the depth of arrays. Fore example, the array mentioned above has 2 depth of arrays.
